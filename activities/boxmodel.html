Aloha!

  • What is the difference between block tags and inline tags?

    • >> Block tags is start on a new line and take up full width (e.g div). While Inline tags is do not start a new line, it only takes as much width as needed (e.g., Span).

  • What is the margin, border, and padding?

    • >> Margin is space outside the border. Border is what surrounds the element, between content and margin. and Padding is space inside the border, around the content.

  • What are units of measurement you can use in CSS?

    • >> Some of the units are ; px: Fixed size (pixels). and % (percentages).

  • When would you use a figure tag?

    • >> Use the the figure tag when we want to group media content with a caption. It helps optimize the SEO.
