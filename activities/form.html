The form tag has an "action" attribute. What does it do?

>> The form action attribute showing what page that going to be shown after the user submit their inputs.

The form tag has a "method" attribute. What does it do?

>> The method meaning what method that we are going to use for request data from the user, there are two type of method : get and post.

What is the difference between a "POST" and a "GET"?

>> The get request means that what user input going to show in the URL, while the post request not. When we wan to do data search we use get request.
