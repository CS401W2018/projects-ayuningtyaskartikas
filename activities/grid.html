What is the main difference between CSS Grid and CSS Flex?.

>> CSS Grid is two dimensions (it has columns and rows), While CSS Flex is only one dimensional. Grid is teritorial parent (the parent is really strict), the parent will tell the kid where to go and the sizes. While Flex is more flexible, the parent will tell the kid where to go, but the kid can decide it own sizes.
