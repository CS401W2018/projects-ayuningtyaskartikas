Beginning CSS

Explain the difference between inline, embedded, and external CSS.

The inline CSS also known as style attributes is when the attributes/property is inside the opening tag, For our final project we not gonna use this.

Embedded CSS also known as style tag, it's put between the open and close style tag

The external CSS is when we use external CSS file and link it to our html file.

Explain why you should only use external CSS.

This is the one that is recommended for us to use, because it's more effective, one css file can be use for the more than one pages that we have (make it cohesive).

What is a selector used for?

The selector is used to select which section from our page that we want the style to be applied (we gonna discuss this more in the next class).
