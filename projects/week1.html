Internet

Internet is a global network where computers can be connected and allows it to communicate and exchange data.

Web

The World Wide Web (WWW) is a system that allow us to accessing information, documents and media on the Internet.

How are the Internet and Web different?

The Internet is the physical network of computers and servers, while the Web is a service that enabling us to access to websites and content on the Internet.

ASCII and Unicode

(American Standard Code for Information Interchange), like its name, ASCII is a standard text encoding, using 7-bit binary codes to represent characters. While Unicode is a more comprehensive system, that aims to cover all the world's writing system.

What is UTF-8

It is a widely used encoding in Unicode, it uses 1 to 4 bytes for each character.

URL

URL (Uniform Resource Locator) is the adresss thatt used to find a source on Internet.

Domain Name

Domain is an address for a website or service on the Internet that is user-friendly (readable by human).

HTTP

HTTP (HyperText Transfer Protocol) is a protocol used for transferring web pages and resources over the Web.

HTTPS

HTTPS (HyperText Transfer Protocol Secure) is similar with http but it aims the secure of transferring web pages process.

IP Address

Internet Protocol (IP) Address is a unique numerical identifier assigned to each device on a network, allowing them to communicate over the Internet.

DNS

(Domain Name System) is a system that translates domain names into IP addresses, so the browser can find and connect to websites.

W3C Recommendations

It is a guidelines or standards developed by the World Wide Web Consortium (W3C) to ensure the Web is accessible, and consistent.

XML, XHTML, HTML, HTML5

XML (eXtensible Markup Language) is used to store and share structured data it is flexible, HTML (HyperText Markup Language) builds the structure of websites, XHTML (eXtensible HyperText Markup Language) is a stricter version of HTML, and HTML5 offers new capabilities for modern web development.
